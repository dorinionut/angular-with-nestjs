# angular-with-nestjs
Small project using NestJS as a Middleware and also as a webserver for an Angular app

## Build Angular app
<pre>
  <code>
    cd client
    ng build --watch
  </code>
</pre>

## Start server
<pre>
  <code>
    cd server
    npm run start
  </code>
</pre>

## Open app
Open your browser and go to http://localhost. The server is running on port 80.
